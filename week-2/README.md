# 📊 Week 2 — SQL Andmete Puhastamine & Andmekvaliteedi Kontroll

## 📌 Projekti eesmärk
Teisel nädalal keskendusin UrbanStyle’i andmebaasi andmekvaliteedi kontrollimisele ja SQL-põhisele andmete puhastamisele Supabase keskkonnas.

---

## 🛠️ Tehtud ülesanded

### ✅ Duplikaatide kontroll
- Duplikaatsete e-mailide leidmine
- `GROUP BY`, `HAVING` ja `ROW_NUMBER()` kasutamine
- Duplikaatide mõju analüüsimine

### ✅ NULL-väärtuste analüüs
- Puuduvate kontaktandmete tuvastamine
- `IS NULL`, `COALESCE()` ja `NULLIF()` kasutamine
- Andmete täielikkuse kontroll

### ✅ Andmevormingute ühtlustamine
- Linnanimede standardiseerimine
- `TRIM()`, `UPPER()`, `LOWER()` ja `INITCAP()` kasutamine
- Ebajärjekindlate kirjaviiside analüüs

---

## 📈 Tulemused
- Leiti duplikaatsed kliendi e-mailid
- Tuvastati puuduvad kontaktandmed
- Standardiseeriti linnanimede formaadid
- Koostati SQL puhastamisskriptid ja andmekvaliteedi raport
