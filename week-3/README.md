# 📊 Week 3 — SQL JOIN Analysis

## 📌 Ülesanne
Analüüsisin UrbanStyle andmeid kasutades erinevaid SQL JOIN päringuid.

## 🛠 Kasutatud tehnikad
- INNER JOIN
- 3 tabeli ühendamine (`sales` + `customers` + `products`)

## 📈 Tulemused
Leidsin suurimad müügid ning sidusin need klientide ja toodetega.  
Analüüs näitas, et kõrgema väärtusega ostud olid seotud premium-toodetega.

## 💡 Järeldus
JOIN päringud aitasid paremini mõista klientide ostukäitumist ja müügitulemusi.

## 🚀 Soovitus
Keskenduda lojaalsetele klientidele ja premium-toodete turundusele.
